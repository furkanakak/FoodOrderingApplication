# Food Ordering Application

## Kodluyoruz &amp; Yemeksepeti Graduation Project

My goal was to create an app that looks like Yemeksepeti.

## Technologies which I used

* MVVM Architecture
* Retrofit - Gson
* Dagger Hilt
* Kotlin Coroutine
* ViewModel - LiveData
* RecyclerView
* Fragments

## Project Structure

* Splash Screen
* Onboarding Screens
* Login Screen
* Register Screen
* Home Screen
* Add Restaurant Screen
* Restaurant Detail Screen
* Add Food Screen
* Food Detail Screen
* Order Basket Screen
* Profile Screen
* Update User Screen
* Last Orders Screen
