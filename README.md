# Foodizone Food Ordering Application

## Kodluyoruz &amp; Yemeksepeti Graduation Project

My goal was to create an app that looks like Yemeksepeti.

## Technologies which I used

 MVVM architecture

 Dagger Hilt

 Retrofit - Gson

 Kotlin Coroutine

 ViewModel - Livedata
 
 Navigation

ViewBinding

 ViewPager2
 
Recyclerview

Fragments

Glide

Lottie
